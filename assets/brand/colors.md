## Typography

### Font Families
- **Headings**: Playfair Display (serif)
- **Body**: Inter (sans-serif)
- **Code/Mono**: JetBrains Mono (monospace)

### Google Fonts Import
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&family=Playfair+Display:ital,wght@0,400;0,600;1,400&family=JetBrains+Mono:wght@400;700&display=swap" rel="stylesheet">
```

## Logo Usage

- Always maintain clear space around the logo
- Minimum size: 40px for icon, 160px for wordmark
- Use dark version on light backgrounds, light version on dark backgrounds
- Never distort, rotate, or change logo colors
